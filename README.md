## Compiling and using the calculator


### Step 1
`sh build_partial.sh`

### Step 2
`build/calculator_partial`

### Step 3
The calculator is ready to handle your statements

`print 1+2;`
outputs
`Calculator output: 3`

#### Or even
`x = 2;`

`y = 3;`

`print x + y;`
outputs
`Calculator output: 5`

### Clean up
